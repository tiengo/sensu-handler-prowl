## sensu-handler-prowl

## Functionality
This is a Sensu Handler to send notifications to a [Prowl](http://www.prowlapp.com/) user, and it follows some patterns from [sensu-plugins](http://sensu-plugins.io).

## Files
 * bin/handler-prowl.rb

## Usage

```
{
  "prowl": {
    "apikey": "abcdefg1234",
    "application": "Sensu"
  }
}
```
